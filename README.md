# Проект по работе с базой данных вакансий с HH.ru
## Описание проекта

### Проект предназначен для получения данных о вакансиях с сайта hh.ru, их хранения в базе данных PostgreSQL и предоставления функционала для анализа этих данных. Система позволяет получать информацию о работодателях и их вакансиях, сравнивать зарплатные предложения и осуществлять поиск по ключевым словам.
Структура проекта
Основные модули:

    main.py - основной исполняемый файл, точка входа в программу

    hh_api.py - модуль для взаимодействия с API hh.ru

    database.py - модуль для управления базой данных PostgreSQL

    vacancy.py - модуль с классами для представления вакансий и работодателей


    requirements.txt - зависимости проекта

    .env - конфигурационные данные (не включен в репозиторий)

Классы:

    HeadHunterAPI - получение данных с API hh.ru

    DBManager - управление базой данных PostgreSQL

    Vacancy - представление вакансии

    Employer - представление работодателя

Функциональность
1. Получение данных с HH.ru

    Получение информации о работодателях по их ID

    Получение списка вакансий для каждого работодателя

    Обработка и валидация данных от API

2. Работа с базой данных PostgreSQL

    Автоматическое создание базы данных и таблиц

    Хранение данных о работодателях и вакансиях

    Обновление существующих записей при конфликтах (UPSERT)

3. Анализ данных (класс DBManager)

    get_companies_and_vacancies_count() - список компаний и количество вакансий

    get_all_vacancies() - все вакансии с указанием компании, зарплаты и ссылки

    get_avg_salary() - средняя зарплата по всем вакансиям

    get_vacancies_with_higher_salary() - вакансии с зарплатой выше средней

    get_vacancies_with_keyword() - поиск вакансий по ключевому слову
